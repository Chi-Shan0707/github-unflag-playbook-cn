# 存在档案

[English](en/06-existence-ledger.md) / 中文

---

> 当一个账号被隐藏，他在平台上的公开存在会突然变得像从未出现过。这个页面记录那些曾经真实存在、真实贡献过的人。

---

## 这是什么

这不是悼念。这是**见证**。

当一个开发者的 GitHub 主页变成 404，他的仓库不再能被搜索到，他的 contributions 从其他项目中消失——他在这个平台上的存在被抹去了。

但他确实存在过。他写过代码，提过 PR，回答过 Issue，帮助过别人。

这个页面就是为这些人留的。

---

## 如何被记录

如果你被 flag甚至suspend，并且无法挽回。你可以联系我们，让我们在这个页面记录你的存在。

### 你可以选择公开多少信息

**最少信息**（一行记录）：
- GitHub 用户名
- 大概被发现 flag 的时间
- 一句话

**完整信息**（一个完整档案）：
- GitHub 用户名
- 地区
- 发现时间
- 主要症状
- 是否恢复
- 一段你想说的话

### 联系方式

| 渠道 | 说明 |
|------|------|
| GitHub Issue | [提交 Issue](../../issues)，标题加 `[存在档案]` 前缀 |
| GitHub Discussion | [参与讨论](../../discussions) |
| Email | 还没想好|

### 隐私承诺

- **默认只公开你同意公开的信息**
- 你可以随时要求删除你的记录
- 你可以要求只公开部分信息
- 不会公开你的邮箱或其他联系方式
