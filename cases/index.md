## 如何提交你的案例

1. 打开案例模板：推荐复制 [GitHub 源码版](https://github.com/Chi-Shan0707/github-unflag-playbook-cn/blob/main/cases/template.md)，也可以先阅读 [网页版模板](https://chi-shan0707.github.io/github-unflag-playbook-cn/cases/template/)
2. 填写你的信息
3. 提交 PR，文件命名为 `<你的用户名>.md`
4. 在 PR 中说明你同意公开这些信息

如果你不想提交 PR，也可以：
- 使用 [案例 Issue 表单](https://github.com/Chi-Shan0707/github-unflag-playbook-cn/issues/new?template=case-report.yml) 填写，我们会帮你整理

## 关于隐私

- 所有案例均经**当事人同意**后公开
- 当事人可以随时要求**修改或删除**自己的案例
- 敏感信息（如工单编号、邮箱）会被脱敏处理
